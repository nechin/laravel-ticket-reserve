# laravel_ticket_test
Laravel ticket handler test task

# Versions
`Laravel Framework 8.12`
`Vue 2.5`
`Bootstrap 4.0`

# Requirements
`php 7.4`

# Installation
1. Clone project
2. Run **composer install**
3. Copy file **.env.example** and rename to **.env**
4. Run **php atrisan key:generate**
5. Run **npm install**
6. Run **npm run dev**
7. Run project
